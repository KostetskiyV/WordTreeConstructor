# WordTreeConstructor

## Условие задачи
Построить бинарное дерево поиска, в вершинах которого находятся слова из текстового файла (предполагается, что исходный файл не содержит повторяющихся слов). Вывести его на экран в виде дерева. Удалить все слова, начинающиеся на указанную букву. Сравнить время удаления слов, начинающихся на указанную букву, в дереве и в файле.
## Исходные данные: 
В параметры запуска программы указывается файл с текстом. В тексте не должно содержаться повторяющихся слов. Словом называется последовательность символов, ограниченная пробелами, символом перевода строки.

Далее  пользователь вводит символ: слова начинающиеся на введённый символ, удаляются.
## Выходные данные:
1. Два дерева слов: образованное из исходного текста и оно же с удалёнными узлами.

2. Новый текстовый файл с удалёнными словами.

3. Замеры времени на удаление слов: в дереве и в файле.
## Задача, реализуемая программой
Поиск и удаление слов из текста по первой букве посредством дерева и файла.
## Способ обращения к программе
Через терминал (app.exe).
## Возможные аварийные ситуации и ошибки пользователя
Неверное количество аргументов при запуске программы. Ошибка открытия тектового файла, ошибка чтения/записи в файл, из терминала. Нехватка памяти для инициализации узла. В тексте обнаружено дублирующееся слово.
